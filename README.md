# Boogeyman 3 - TryHackMe Write-up

This repository contains my write-up for the TryHackMe **Boogeyman 3** room.

The investigation focused on analyzing attacker activity across multiple Windows hosts using Elastic/Kibana logs. The write-up covers PowerShell execution, remote share enumeration, credential discovery, lateral movement, hash dumping with Mimikatz, DCSync activity, and ransomware download/execution evidence.
